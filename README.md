# Building

For `x86_64`-based machines running Unix-like OSes:
```bash
❯❯ cargo b --release --features x86
```

For Apple Silicon or `aarch64`-based machines running Unix-like OSes:
```bash
❯❯ cargo b --release --features aarch64
```
